# Future Version 2.0

During the development of HairFlow, several ideas came up that could improve the application in the future. Due to time constraints, these features were not implemented in version 1.0.

## Better appointment calendar

Instead of showing appointments in a table, it would be interesting to create a weekly calendar.

Each appointment could appear as a block in the corresponding day and hour, making it easier for employees to organise their schedule.

## Appointment rescheduling

Currently appointments can only be created, edited or deleted.

A future version could allow employees to suggest a different appointment time.

The client would then receive the request and decide whether to accept or reject the new time.

## Employee working schedule

Employees could define their own working hours.

For example:

- Monday to Friday
- Working hours
- Lunch break
- Vacation days

The system would automatically avoid creating appointments outside these hours.

## Client improvements

Clients could have access to more information, such as:

- Appointment history
- Favourite services
- Personal profile
- Previous appointments

## Notifications

The system could send email notifications when:

- An appointment is created.
- An appointment is cancelled.
- An appointment is rescheduled.
- An appointment is close to its scheduled time.

## Dashboard improvements

The administrator dashboard could display some statistics, for example:

- Total appointments
- Number of clients
- Number of employees
- Most requested services

## Better search and filters

Adding search bars and filters would make it easier to find employees, services and appointments.

## Mobile improvements

Although the application already works on different screen sizes, the interface could be improved for mobile devices.

## Final Note

The current version focuses on implementing the core functionality of the application.

The ideas presented above are intended as possible improvements that could make HairFlow more complete and closer to a real salon management system.
